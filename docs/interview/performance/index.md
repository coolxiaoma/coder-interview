# 性能优化面试题

性能优化是前端开发中重要的一部分，掌握性能优化技术可以帮助开发者创建更加流畅、响应更快的应用。本章节收集了前端性能优化相关的面试题，帮助您更好地准备性能优化面试。

## 核心知识点

- **页面加载优化**：减少资源大小、减少HTTP请求、使用CDN、预加载等
- **渲染优化**：减少DOM操作、使用虚拟DOM、优化CSS选择器、使用requestAnimationFrame等
- **JavaScript优化**：减少重绘和回流、使用事件委托、优化事件处理等
- **资源优化**：图片优化、字体优化、代码分割等
- **缓存优化**：浏览器缓存、Service Worker缓存、HTTP缓存等
- **网络优化**：使用HTTP/2、HTTP/3、压缩资源、减少请求头等

## 面试准备建议

1. **理解性能指标**：了解关键性能指标如FCP、LCP、CLS、TTI等
2. **掌握优化策略**：熟悉各种性能优化策略及其适用场景
3. **使用性能工具**：熟练使用Chrome DevTools、Lighthouse等性能分析工具
4. **实践经验**：在实际项目中应用性能优化技术，积累经验
