# 工程化面试题

前端工程化是现代前端开发的重要组成部分，掌握前端工程化知识可以帮助开发者提高开发效率、确保代码质量、优化部署流程。本章节收集了前端工程化相关的面试题，帮助您更好地准备工程化面试。

## 核心知识点

- **构建工具**：Webpack、Vite、Rollup、Parcel 等
- **包管理工具**：npm、yarn、pnpm 等
- **版本控制**：Git 相关知识，包括分支管理、Git 工作流等
- **CI/CD**：持续集成和持续部署，包括 GitHub Actions、GitLab CI、Jenkins 等
- **模块化**：ES Modules、CommonJS、AMD、CMD 等
- **代码规范**：ESLint、Prettier、Stylelint 等
- **测试**：单元测试、集成测试、E2E 测试，包括 Jest、Cypress、Vitest 等
- **性能优化**：构建优化、代码分割、懒加载等
- **Monorepo**：Lerna、Nx 等

## 面试准备建议

1. **掌握核心工具**：熟练掌握 Webpack、Vite 等构建工具的使用和配置
2. **了解 Git 工作流**：掌握常见的 Git 工作流，如 Git Flow、GitHub Flow 等
3. **学习 CI/CD**：了解 CI/CD 的基本概念和常用工具
4. **代码规范**：掌握 ESLint、Prettier 等代码规范工具的使用
5. **测试知识**：了解前端测试的基本概念和常用测试框架
