# HTML/CSS面试题

HTML和CSS是前端开发的基础，掌握HTML和CSS的核心概念对于前端开发者来说至关重要。本章节收集了HTML/CSS相关的面试题，帮助您更好地准备HTML/CSS面试。

## 核心知识点

- **HTML基础**：HTML5 新特性、语义化标签、表单、Canvas 等
- **CSS基础**：CSS选择器、盒模型、布局（Flexbox、Grid、Float、Position 等）、单位、颜色等
- **CSS进阶**：CSS预处理器（Sass、Less、Stylus）、CSS Modules、CSS-in-JS 等
- **响应式设计**：媒体查询、移动优先设计等
- **CSS动画**：Transition、Animation、Transform 等
- **CSS新特性**：CSS变量、@layer、@container 等
- **浏览器兼容性**：浏览器兼容性问题及解决方案

## 学习资源

- [MDN Web Docs](https://developer.mozilla.org/zh-CN/)：HTML和CSS的官方文档
- [CSS-Tricks](https://css-tricks.com/)：CSS相关的教程和技巧
- [HTML5 Rocks](https://www.html5rocks.com/zh/)：HTML5相关的教程和资源
- [CSS Secrets](https://book.douban.com/subject/26752777/)：CSS高级技巧相关的书籍

## 面试准备建议

1. **掌握核心概念**：熟练掌握HTML和CSS的核心概念，如盒模型、布局、选择器等
2. **实践项目**：通过实际项目练习，加深对HTML和CSS的理解和应用
3. **关注新特性**：了解HTML5和CSS3的新特性
4. **学习最佳实践**：掌握HTML和CSS的最佳实践，如语义化、性能优化等
5. **解决兼容性问题**：了解常见的浏览器兼容性问题及解决方案
