# Vue 面试题

Vue 是一套用于构建用户界面的渐进式 JavaScript 框架，易于学习，灵活多变，深受前端开发者喜爱。本章节收集了 Vue 相关的常见面试题，帮助您更好地准备 Vue 相关的面试。

## 目录结构

- [Vue 基础](./basic.md)：Vue 基础概念和核心特性
- [高频面试题](./high-frequency.md)：Vue 高频面试题，帮助您准备面试
- [无答案面试题（用于模拟面试）](./no-answer.md)：Vue 无答案面试题，模拟真实面试场景

## 学习资源

- [Vue 官方文档](https://cn.vuejs.org/)：Vue 官方文档，是学习 Vue 的最佳资源
- [Vue Mastery](https://www.vuemastery.com/)：Vue 视频教程

## 面试准备建议

1. **掌握核心概念**：深入理解 Vue 的核心概念，如组件、指令、生命周期、响应式原理等
2. **实践项目**：通过实际项目练习，加深对 Vue 的理解和应用
3. **阅读源码**：阅读 Vue 源码，了解其内部实现原理
4. **关注 Vue 3**：了解 Vue 3 的新特性和改进
5. **学习生态工具**：掌握 Vue Router、Vuex/Pinia 等生态工具
