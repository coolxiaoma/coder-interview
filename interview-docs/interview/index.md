# 面试题分类

欢迎来到前端面试题库！这里包含了前端开发领域常见的面试题，按不同分类进行整理，方便您学习和查阅。

## 分类说明

- **HTML/CSS**：HTML结构、CSS样式、布局、响应式设计等相关面试题
- **JavaScript**：JavaScript基础、高级特性、异步编程、DOM操作等相关面试题
- **浏览器原理**：浏览器渲染机制、事件循环、同源策略、Cookie等相关面试题
- **前端框架**：Vue、React、Angular等主流前端框架相关面试题
- **性能优化**：页面加载优化、渲染优化、资源优化等相关面试题
- **工程化**：构建工具、版本控制、CI/CD、模块化等相关面试题
- **算法**：前端开发中常见的算法题和数据结构
- **网络**：HTTP/HTTPS、TCP/IP、跨域、WebSocket等相关面试题
- **设计模式**：前端开发中常用的设计模式
- **其他**：前端安全、移动端开发、TypeScript等相关面试题

## 使用方法

1. 在左侧侧边栏选择您感兴趣的分类
2. 浏览该分类下的面试题
3. 使用顶部的搜索框搜索特定的面试题
4. 点击面试题标题查看详细内容
